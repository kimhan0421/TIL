3. Loading State
</h1>

```ts
type LoadingState = {
	state: 'loading';
};

type SuccessState = {
	state: 'success';
	response: {
		body: string;
	};
};

type FailState = {
	state: 'fail';
	reason: string;
};

type ResourceLoadState = LoadinfState | SuccessState | FailState;
```
