<h1 align="center">
Type alias
</h1>

- typescript의 장점!
- 새로운 타입을 정할 수 있다!

```jsx
type Text = string;
const name: Text = 'Lily';
const address: Text = 'korea';
type Num = number;
```

```jsx
type Student = {
  name: stinrg
  age: number
}
```

=> 이 경우, Student에는 name과 age만 가능!

ex)

```jsx
const student: Student = {
  name: 'lily'
  age: '10'
}
```
