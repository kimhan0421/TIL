<h2>Object Oriented Programming</h2>

- 객체들끼리 의사소통하도록 코딩
- object 단위 (하나의 object에는 data, function이 존재 가능.)
- productivity, higher-quality, faster

<h3>object</h3>

- object의 속성을 가지고 있는 data
- 코드를 수행 할 수 있는 function

ex) MediaPlayer
music - data
play - function
stop - function

ex) object로 관리 할 수 있음

- Error

- Event
