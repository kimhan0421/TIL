<h1 align="center">
inference(타입 추론)</h1>

- 타입이 알아서 정해주는 경우

```ts
let text = 'hello';
text = 1; //error
```

=> text 는 선언과 동시에 타입을 할당. 따라서 1이라고 선언 했을 때 에러!

```ts
function print(message = 'hello') {
	console.log(message);
}

print('hello');
print(1); //error
```

<h3 align="center">
inference(타입 추론) - 비추! 타입을 명시!</h3>

=> 간단한 것은 추론

=> 가독성이나 협업 등을 위해서 타입 명시!
