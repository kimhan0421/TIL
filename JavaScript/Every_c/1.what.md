자바스크립트로 뭘 할 수 있나요?
==============================

night 버튼 => input으로 줄 수 있음 + 속성 type 

value = ‘night’ 으로 이름 줄 수 있음. 

onclick이라는 속성 안에 자바스크립트를 넣어줌 => 사용자가 버튼을 클릭했을 때 실행시킴. 

전체 body => CSS를 넣음. night과 day버튼을 누름에 따라 값이 바뀜

### 자바스크립트
1. 사용자와 상호작용함.
2. 웹브라우저는 바꿀 수 없다. 그러나 자바스크립트에 따라서 디자인을 바꿀 수 있음.
